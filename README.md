# 잡히면 #윤밥사

팩맨풍 2D 아케이드 웹게임 프로토타입입니다.

## 게임 방법

- 방향키 또는 WASD로 윤상용을 움직입니다.
- 던전의 인세 코인을 모두 먹으면 승리합니다.
- 정대원(곰)이나 정보람(다람쥐)에게 잡히면 `#윤밥사` 돌글씨가 떨어지고 먹은 인세를 모두 뱉습니다.
- 모바일에서는 화면의 방향 버튼을 사용할 수 있습니다.

## GitHub Pages 주소

저장소 이름이 `yoonbapsa-game`이고 계정이 `polargomz`라면 배포 주소는 다음과 같습니다.

`https://polargomz.github.io/yoonbapsa-game/`

## 배포 설정

저장소의 **Settings → Pages → Build and deployment → Source**에서 **GitHub Actions**를 선택합니다. 이후 `main` 브랜치에 푸시하면 자동 배포됩니다.
